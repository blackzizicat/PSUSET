# PSUSET（Printer/Scanner Usage Statistics Extraction Tool）
## はじめに
ここでは，月初のプリンタ・スキャナ利用統計レポートの作成手順について記載する。

### 工程
毎月最初の営業日に実施する。
1. プリンタサーバからデータを取得
2. データをExcelに入れて集計
3. 結果を報告

メールレポート作成まで自動化したので，まずその利用手順を記載する。
（DockerデスクトップとGitがインストールされている前提）

## 事前準備(初回のみ)
作業フォルダ内でクローンしてビルドする。
```
> git clone https://github.com/imshinjo/PSUSET.git
> cd .\PSUSET\
PSUSET> docker build -t psuset .
```

### 統計記入ファイルの記入方法
#### 月のセル
統計記入ファイルの各月のセルで，下記のようにして値を取得できる。
詳しくはExcelの使い方になるので割愛する。

- ロビープリンタ利用統計.xlsx
`=VLOOKUP([@HOST],'最新の機器カウンターレポート(JA)_YYYYMMDD_xxxxxx.csv'!$G$8:$L$99,5,FALSE)`

- 教員カラープリンタ印刷統計.xlsx
モノクロ
`=VLOOKUP([@HOST],'最新の機器カウンターレポート(JA)_YYYYMMDD_xxxxxx.csv'!$G$8:$L$99,5,FALSE)`
カラー
`=VLOOKUP([@HOST],'最新の機器カウンターレポート(JA)_YYYYMMDD_xxxxxx.csv'!$G$8:$L$99,6,FALSE)`

- 教室等モノクロプリンタ印刷統計.xlsx
`=VLOOKUP([@HOST],'最新の機器カウンターレポート(JA)_YYYYMMDD_xxxxxx.csv'!$G$8:$L$99,5,FALSE)`

- Ricohスキャナ統計.xlsx
範囲をホスト名からにすると，36列目カラー，37列目モノクロ
モノクロ
`=VLOOKUP([@HOST],'機能×カラー別集計レポート 月毎(JA)_YYYYMMDD_xxxxxx.xls'!$B$7:$AL$83,37,FALSE)`
カラー
`=VLOOKUP([@HOST],'機能×カラー別集計レポート 月毎(JA)_YYYYMMDD_xxxxxx.xls'!$B$7:$AL$83,36,FALSE)`

地味に重要なのが，これで値が埋まっただけで終わってはいけない。
セルに数式が入っている状態では，後述するグラフレポートで利用できないので，
セルをコピー＆ペーストして「値のみ貼り付け」を実行して，文字列型にする必要がある。
具体的な操作はExcelの使い方になるので割愛する。


#### 右端のセル
月のセルの右端（枠外）に一列，数値が並んでいる。
これは先月の一か月間でプリントやスキャンされた枚数を示している。

- プリンタの集計レポートの数値は「累計」のため，
先月にとれた数値から，ひとつ前の数値を引き算して，先月1か月の枚数をセルに表示させる。
- スキャナの集計レポートの枚数は，そのまま先月の数値をセルに表示させる。
- 数値が多いセルに赤い色がつくので，後述する統計レポート（メール）の文章を作る際に参照する。
- このセルは数式のままで良い。（値のみ貼り付けしなくてよい）


#### Zabbixの確認
値が取れていないものはZabbixで確認する。
1. Zabbixダッシュボード 左ペインの「最新データ」をクリックする。
2. 「名前」欄にプリンタ名を入力して検索する。
3. 「カラースキャン枚数」または「モノクロスキャン枚数」の「グラフ」をクリックする。
4. 右上「表示形式」で「値」を選択して，期間を「先月」にする。
5. 表示された数値を記録する。表示されない場合はゼロとする。

グラフの作成といってもこれだけのことである。

### レポート作成
次にグラフレポートに表示されたグラフを見て，その状況を文章にする必要がある。
文章のフォーマットは下記のように決まっている。

- 〇〇には増減（または「利用なし」）xxには場所，□□には枚数を書く
- 2行にする
- 7，8月はまとめて作成する

```
・プリンタ，スキャナ
当月は教室プリンタは ○○，
ロビープリンタは ○○，
教員室プリンタはカラーが ○○，白黒は○○。
Xeroxスキャナはカラーが ○○，白黒は○○。
ロビープリンタのスキャナはカラーが ○○，白黒は ○○。
教室プリンタのスキャナはカラーが ○○，白黒は ○○。

特に目立った利用は，
教室プリンタ（ xx → □□枚），
教員室プリンタ（ xx → カラー: □□枚，白黒: □□枚），
ロビープリンタ（ xx → □□枚），
ロビープリンタのスキャナ（ xx → □□枚），
教室プリンタのスキャナ（ xx → □□枚）だった。
```
(「2行にする」ルールがあるが，わかりやすさのために改行している。)

この通り，印刷枚数，スキャン枚数の増減を報告する前半と，
特に利用が多かった部分を報告する後半に分かれている。

- 前半に関して，具体的な数値は統計レポートやグラフレポートにあるので，
ここでは，増加した，減少した，横ばい，などの表現をするにとどめる。

- 後半に関して，戦術した統計レポート（Excel）の右端のセルで
赤く色がついた部分の，建屋・部屋の名前と，枚数をここで報告する。

- 部屋の名前について，部屋番号のほかに総称がついている場合は含める。
例えば，
「10号館10301」ではなく「10号館10301オープンパソコンルーム」とする。


これらを踏まえると下記のようになる。
```
・プリンタ，スキャナ
当月は教室プリンタ・ロビープリンタともに大幅に減少，教員室プリンタもカラー・白黒ともに大幅に減少した。Xeroxスキャナはカラー・白黒ともに利用なしだった。ロビープリンタのスキャナは，カラー・白黒ともに大幅に減少した。教室プリンタのスキャナのカラーは変化なし，白黒は利用なしだった。
特に目立った利用は，教室プリンタ（10号館10301オープンパソコンルーム→522枚），教員室プリンタ（15号館15211研究談話室→カラー 8,184枚，白黒→4,162枚），ロビープリンタ（10号館2階ロビー右→1,612枚），ロビープリンタのスキャナ（万有館4階サイエンスラウンジ→77枚），教室プリンタのスキャナ（10号館10301オープンパソコンルーム→2枚）だった。
```
これは冬休みに入った月のレポートで，冬休みに利用が減少するのは当然だが，
全部「減少した」という情報のみで問題なかった。
